# **App Name**: StreamFlow

## Core Features:

- Music Library: Browse and search for local music files by songs, albums, playlists, and artists. Display all local .mp3 and .wav audio files in an easily navigable interface.
- Audio Playback: Play local .mp3 or .wav files with essential controls such as play, pause, skip, seek, repeat, and shuffle. Provide a seamless listening experience using native audio playback capabilities.
- Playlist Management: Create, edit, and save playlists to organize music. Allow users to add and remove tracks from their custom playlists for personalized listening.
- Basic Metadata Handling: Automatically scan .png and .jpg cover art for songs in local directories
- UI/UX Design: Develop a responsive, modern, and intuitive user interface featuring lists, grids, a mini-player, and a full-screen player view for optimal user engagement and ease of navigation.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to evoke a sense of calm and tranquility, suitable for a relaxed listening experience.
- Background color: Very light blue (#EBF4FA), almost white, maintaining a clean and airy aesthetic.
- Accent color: Deep blue (#3F51B5) for interactive elements, providing contrast and focus.
- Body and headline font: 'PT Sans', a versatile sans-serif, ensuring readability and a modern feel.
- Use minimalist icons for playback controls and navigation, ensuring clarity and ease of use.
- Employ a grid layout for albums and playlists, offering a visually appealing and organized display.
- Incorporate subtle animations for transitions and interactions to enhance the user experience without being intrusive.
