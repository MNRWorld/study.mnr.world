# স্টাডি - আপনার পড়াশোনার সঙ্গী

"স্টাডি" একটি আধুনিক শিক্ষা প্ল্যাটফর্ম যা শিক্ষার্থীদের একাডেমিক লক্ষ্যে পৌঁছাতে সাহায্য করার জন্য তৈরি করা হয়েছে। এই অ্যাপটি Next.js, React, এবং Tailwind CSS ব্যবহার করে নির্মিত হয়েছে, যা একটি দ্রুত এবং প্রতিক্রিয়াশীল ব্যবহারকারীর অভিজ্ঞতা নিশ্চিত করে।

## ✨ বৈশিষ্ট্য (Features)

- **আধুনিক ডিজাইন**: একটি নান্দনিক এবং ব্যবহারকারী-বান্ধব ইন্টারফেস।
- **প্রশ্ন ব্যাংক**: বিভিন্ন বিষয় এবং পরীক্ষার জন্য প্রশ্নপত্রের বিশাল সংগ্রহ।
- **ভর্তি ক্যালেন্ডার**: বিশ্ববিদ্যালয় এবং কলেজের ভর্তির সময়সূচী ট্র্যাক করার জন্য একটি ক্যালেন্ডার।
- **কোর্সসমূহ**: বিভিন্ন একাডেমিক স্তরের জন্য কোর্স এবং টিউটোরিয়াল।
- **ডার্ক/লাইট মোড**: ব্যবহারকারীর পছন্দ অনুযায়ী থিম পরিবর্তনের সুবিধা।

## 🛠️ ব্যবহৃত প্রযুক্তি (Tech Stack)

- **ফ্রেমওয়ার্ক**: [Next.js](https://nextjs.org/)
- **ল্যাঙ্গুয়েজ**: [TypeScript](https://www.typescriptlang.org/)
- **UI লাইব্রেরি**: [React](https://react.dev/)
- **স্টাইলিং**: [Tailwind CSS](https://tailwindcss.com/)
- **UI কম্পোনেন্ট**: [ShadCN UI](https://ui.shadcn.com/)
- **অ্যানিমেশন**: [Framer Motion](https://www.framer.com/motion/)
- **আইকন**: [Lucide React](https://lucide.dev/)

## 🚀 শুরু করুন (Getting Started)

প্রথমে, ডেভেলপমেন্ট সার্ভার চালু করুন:

```bash
npm run dev
```

আপনার ব্রাউজারে [http://localhost:9002](http://localhost:9002) ভিজিট করে অ্যাপটি দেখতে পারবেন।

`src/app/page.tsx` ফাইলটি পরিবর্তন করে আপনি লাইভ পরিবর্তন দেখতে পারেন।
